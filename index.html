Sushant Mittal Mini Projects

Vanilla JS frontend projects

Fun games and projects developed using simple front end technologies like HTML CSS and Vanilla JS. Some of these also incorporate the use of APIs. All projects are hosted via GitHub.

Hangman Game. Guess letters of the hidden word to save the hangman. Wrong guesses lead to hanging of the hangman.

Typing tester. Generates random words which are then to be typed by the user. Contains three difficulty modes : Easy medium and hard.

Simon Game. A common memory test game where the player is required to remember a sequence of colors generated randomly.

Dice Decider. Helps make a decision when the players can't reach a conclusion using two standard dice. Player with higher roll wins.

Virtual drumkit. Plays various components of the drums by listening to keystrokes. Also generates corresponding sounds.

Meal Finder. Finds meals from a static meal dataset based on the user's input. Makes use of the themealdb.com API

Breathing regulator. Helps the user regulate their breathing by prompting appropriate breathe in and breathe out times.

Find Lyrics. Takes in user input as a unique song and returns the lyrics. Makes use of the lyrics.ovh API

Expenditure tracker. Enables user to add and remove transactions to show net spending. Transactions stord in localstorage of browser.
